# Dinamica pe biti

## Probleme de interviuri
1. Numar maximal de puncte vazute de o sursa de lumina de unghi `r`.
2. Problema determinarii legaturilor electrice dintr-un bloc folosind cat mai putine urcari cu
liftul.
